## ENGLISH

### 1. Introduction

This Privacy Policy describes how "Radio Amateur Digital ID Query" (hereinafter referred to as "the Application") collects, uses, and protects user information. The Application is developed by ALG Yazılım & Elektronik Inc. and is designed for radio amateurs to query DMR, NXDN, Repeater IDs, and APRS data.

**This application is 100% free and completely ad-free, forever.**

### 3. Data Collection and Usage

#### 3.1 Data We Collect

The Application does NOT collect, store, or transmit any personal user data to our servers. All data operations are performed as follows:

- **Search Queries:** User-entered callsigns and IDs are sent directly to third-party APIs (RadioID.net and APRS.fi) for query purposes only.
- **Local Cache:** Search results are stored locally on the user's device using Android SharedPreferences for offline access.
- **No Analytics:** We do not use any analytics, tracking, or advertising services. The application is completely ad-free.
- **No User Accounts:** The Application does not require user registration or login. It is 100% free with no subscriptions or in-app purchases.

#### 3.2 Third-Party Data Sources

The Application retrieves public radio amateur information from the following sources:

**RadioID.net API:**
- Purpose: Query DMR, NXDN, and Repeater IDs
- Data Retrieved: Callsign, Name, City, Country, Radio ID, Last Talk Group, Frequency, Trustee
- Privacy Policy: https://radioid.net
- Data Type: Publicly available radio amateur database

**APRS.fi API:**
- Purpose: Query APRS station information
- Data Retrieved: Callsign, Latitude, Longitude, Comment, Symbol
- Privacy Policy: https://aprs.fi
- Data Type: Publicly available APRS network data

**Important:** All data retrieved from these APIs is publicly available information that radio amateurs have voluntarily submitted to these services.

### 4. Data Storage

#### 4.1 Local Storage (On-Device)

- **Cache Duration:** 
  - DMR/NXDN/Repeater queries: 24 hours
  - APRS queries: 1 hour
- **Storage Method:** Android SharedPreferences (encrypted by Android OS)
- **Storage Location:** Application's private directory on device
- **User Control:** Users can clear cache by uninstalling the application

#### 4.2 No Cloud Storage

The Application does NOT store any user data on remote servers or cloud services.

### 5. Permissions

The Application requires the following Android permissions:

#### 5.1 INTERNET Permission
- **Purpose:** To query radio amateur data from RadioID.net and APRS.fi APIs
- **Usage:** Only used when user initiates a search query
- **Data Transmission:** Search queries are sent directly to third-party APIs; no data is sent to our servers

#### 5.2 No Other Permissions Required

The Application does NOT require:
- Location access
- Camera access
- Contacts access
- Storage access (except app-private storage)
- Phone state access
- Any other sensitive permissions

### 6. Data Sharing

#### 6.1 No Data Sharing with Third Parties

We do NOT share, sell, or transmit user data to any third parties except:
- Direct API queries to RadioID.net and APRS.fi (as described in Section 3.2)
- User-initiated sharing via Android's share functionality (WhatsApp, Telegram, etc.)

#### 6.2 User-Initiated Sharing

The Application provides features to share query results:
- **Share Button:** Users can voluntarily share results via Android's native share dialog
- **QR Code Generation:** QR codes are generated locally on the device and are not transmitted anywhere
- **User Control:** All sharing is explicitly initiated by the user

### 7. Data Security

- **Local Encryption:** Data cached on device is protected by Android OS security
- **HTTPS Communication:** All API communications use HTTPS protocol
- **No Authentication Required:** Application does not store passwords or authentication tokens
- **Minimal Data Retention:** Cached data expires automatically (24 hours for IDs, 1 hour for APRS)

### 8. Children's Privacy

The Application does not knowingly collect information from children under 13 years of age. The Application is designed for licensed radio amateurs who are typically adults or supervised minors.

### 9. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Users will be notified of significant changes through application updates. Continued use of the Application after changes constitutes acceptance of the updated policy.

### 10. User Rights

Users have the right to:

- **Deletion:** Delete all cached data by clearing app data or uninstalling
- **Opt-Out:** Stop using the Application at any time
- **Contact:** Reach out to us with privacy concerns at info@algyazilim.com

Uygulama, aşağıdaki kaynaklardan halka açık radyo amatörü bilgilerini alır:

**RadioID.net API:**
- Amaç: DMR, NXDN ve Röle ID'lerini sorgulama

**APRS.fi API:**
- Amaç: APRS istasyon bilgilerini sorgulama
- Alınan Veriler: Çağrı işareti, Enlem, Boylam, Yorum, Sembol
- Gizlilik Politikası: https://aprs.fi
- Veri Türü: Halka açık APRS ağ verileri

**Önemli:** Bu API'lerden alınan tüm veriler, radyo amatörlerinin bu hizmetlere gönüllü olarak gönderdikleri halka açık bilgilerdir.
